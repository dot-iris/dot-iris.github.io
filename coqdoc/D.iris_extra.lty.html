D.iris_extra.lty

Formalize infrastructure for semantic types; not shown in the paper.

In Iris one represents semantic types as persistent Iris predicates on values. Since D* syntactic types can contain variables ranging on values, semantic types take a value substitution as argument.
Using Autosubst, we can define substitution on semantic types by precomposition: τ.|[s] = λ ρ, τ (ρ >> s) .
All this infrastructure works for arbitrary languages implementing ValuesSig.

    (∀ args ρ, olty_car τ1 args ρ = olty_car τ2 args ρ) →
    τ1 = τ2.
  Proof. intros × Heq; f_ext ⇒ args; f_ext ⇒ ρ. apply lty_eq, Heq. Qed.

  Lemma olty_eq' τ1 τ2: olty_car τ1 = olty_car τ2 → τ1 = τ2.
  Proof.
    intros Heq; apply olty_eq ⇒ args ρ.
    apply (f_equal_dep _ _ ρ
              (f_equal_dep _ _ args Heq)).
  Qed.

  Lemma lift_olty_eq {τ1 τ2 : oltyO Σ} {args ρ v} :
    τ1 ≡ τ2 → τ1 args ρ v ≡ τ2 args ρ v.
  Proof. apply. Qed.

    (∀ args ρ v, T1 args ρ v ≡ T2 args ρ v) ⊣⊢@{iPropI Σ} (T1 ≡ T2).
  Proof.
